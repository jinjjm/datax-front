# Ant Design Pro

This project is initialized with [Ant Design Pro](https://pro.ant.design). Follow is the quick guide for how to use.

## Environment Prepare

Install `node_modules`:

```bash
npm install
```

or

```bash
yarn add xxx
```

### Start project

```bash
yarn start
```

### Build project

```bash
yarn build
```

### Check code style

```bash
npm run lint
```

### Test code

```bash
npm test
```

## 提交Git仓库
```bash
git add .
git commit -m '描述' --no-verify # --no-verify忽略检查
git pull
git push
```
